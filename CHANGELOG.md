# Changelog

All notable changes to mac-editor will be documented in this file.

## [v0.3.0] — 2026-05-29

### Added
- Markdown 实时预览 — 编辑 .md 文件时点击工具栏眼睛按钮，右侧显示渲染后的预览面板（左右分屏）
- 本地图片支持 — 预览中绝对路径和相对路径图片通过 Rust 读取为 base64 data URL 显示
- GitHub 风格 Markdown 渲染样式（标题、代码块、表格、引用块、任务列表等）

## [v0.2.0] — 2026-05-22

### Added
- 按行去重功能（⌘⇧D），有选区去重选中行，无选区全文去重
- Tab 拖拽排序，蓝色插入指示条，Chrome 风格标签视觉（活跃 tab 浮起、非活跃透明融合）
- 关闭保存确认 — 关闭窗口时逐个询问每个未保存文件，保存/不保存/取消三按钮（Excel 风格）
- 单 tab 关闭确认 — 点击 × 或 Cmd+W 关闭已修改文件时弹出保存询问
- 新建 tab 顺序命名："未命名"→"未命名1"→"未命名2"...
- 工具栏：按行去重按钮、列选择按钮

### Changed
- 工具栏去掉字号下拉框和语言图标，语言名称统一英文
- Tab 修改标记圆点缩小至 7px

### Fixed
- 保存新文件时取消另存为对话框后，tab 不应被关闭
- Cmd+W 关闭已修改 tab 时绕过保存确认
- 批量关闭过程中仍可触发单个 tab 关闭导致对话框叠加
- CRLF 换行符在不可见字符视图中检测错误

## [v0.1.0] — 2026-05-19

### Added
- 初始版本：Tauri 2 + React 19 + CodeMirror 6 桌面编辑器
- 多标签页编辑，30+ 语言语法高亮
- 文件打开/保存/另存为，编码自动检测（chardetng）
- 搜索替换面板（正则、大小写、转义序列、标记所有）
- 不可见字符显示切换
- 跳转到指定行
- 多光标编辑（Cmd+Click、列选择）
- 最近文件列表（localStorage 持久化）
- 文件拖放打开
- 快捷键支持（新建/打开/保存/关闭/搜索/字体缩放/切换 tab）
